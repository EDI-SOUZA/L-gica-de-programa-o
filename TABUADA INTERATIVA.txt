CRIAR UMA ENTRADA AO USUARIO PEDINDO PARA O MESMO DIGITAR UM NÚMERO
CASO O USUARIO DIGITAR O NUMERO 0
SERA MOSTRADA A SEGUINTE MENSAGEM DE FINALIZAÇAO DO PROGRAMA ("PROGRAMA ENCERRADO".) E NÃO SERÁ PEDIDO MAIS UM NÚMERO AO USUÁRIO. 
ENQUANTO O USUARIO DIGITAR UM NUMERO DIFERENTE DE 0, VAI CONTINUAR SENDO PEDIDO UM NUMERO AO MESMO
PARA O NUMERO DIGITADO, MOSTRAR NA TELA A TABUADA DO NUMERO DIGITADO ATÉ QUE O MESMO DIGITE 0 PARA ENCERRAR O PROGRAMA
